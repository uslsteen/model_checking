\begin{frame}{Вкратце об электромагнитных волнах}

В общем случае волновое уравнение - это

\[  \frac{\partial^{2} S}{\partial t^2} = v^{2} \frac{\partial^{2} S}{\partial x^{2}}  \]


Пусть отсутствуют токи проводимости, свободные заряды.

Тогда из уравнений Максвелла можем получить:

\[    \frac{\partial^{2} \textbf{B}}{\partial t^2} = \frac{c^{2}}{\epsilon\mu} \Delta \textbf{B}\]

\[    \frac{\partial^{2} \textbf{E}}{\partial t^2} = \frac{c^{2}}{\epsilon\mu} \Delta \textbf{E}\]



\end{frame}
